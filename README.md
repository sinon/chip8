# Chip-8 Emulator

[![codecov](https://codecov.io/gh/sinon/chip8/graph/badge.svg?token=9QYDO9E4RU)](https://codecov.io/gh/sinon/chip8)

Spec: <http://devernay.free.fr/hacks/chip8/C8TECH10.HTM>

## Run ROMs

### Pong

`cargo run pong`

Controls:
- Player 1 Up: 1
- Player 1 Down: Q
- Player 2 Up: 4
- Player 2 Up: R
- Quit: Esc


### Guess

`cargo run guess`

### Maze

`cargo run maze`
